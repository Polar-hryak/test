# Инструкции по работе с SSH и GitHub

## 1. Как создать SSH ключ

1. Откройте терминал (или командную строку).
2. Введите следующую команду, чтобы создать новый SSH ключ:
   ```bash
   ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

   - Замените "your_email@example.com" на ваш адрес электронной почты, который вы используете для GitHub.
3. Когда вас попросят указать путь для сохранения ключа, нажмите Enter, чтобы использовать путь по умолчанию (~/.ssh/id_rsa).
4. Вам будет предложено установить пароль для ключа. Это опционально, но рекомендуется добавить пароль для большей безопасности.
5. Ваш SSH ключ будет создан. Теперь вы можете найти его в директории ~/.ssh/:

   ls ~/.ssh

## 2. Как добавить ключ в аккаунт на GitHub

1. Скопируйте содержимое вашего публичного SSH ключа в буфер обмена. Для этого выполните следующую команду в терминале:

   pbcopy < ~/.ssh/id_rsa.pub  # Для macOS
   cat ~/.ssh/id_rsa.pub        # Для Linux/Windows, затем вручную скопируйте содержимое

2. Перейдите на GitHub и войдите в свой аккаунт.
3. В правом верхнем углу страницы нажмите на иконку профиля и выберите Settings.
4. В левом меню выберите SSH and GPG keys.
5. Нажмите на New SSH key или Add SSH key.
6. Вставьте скопированный SSH ключ в поле Key. Добавьте заголовок в поле Title (например, "My Laptop SSH Key").
7. Нажмите Add SSH key для сохранения.

## 3. Как склонировать репозиторий

1. Перейдите на страницу репозитория на GitHub, который вы хотите склонировать.
2. Нажмите на зеленую кнопку Code и выберите SSH.
3. Скопируйте ссылку, которая отображается (например, git@github.com:username/repo.git).
4. Откройте терминал и выполните следующую команду для клонирования репозитория:

   git clone git@github.com:username/repo.git

   - Замените username/repo.git на соответствующий путь к вашему репозиторию.
5. Репозиторий будет склонирован в текущую директорию.
